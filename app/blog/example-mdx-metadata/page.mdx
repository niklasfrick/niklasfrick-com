# Wie man Metadaten aus MDX für Next.js SEO exportiert

<BlogDateDisplay date="2024-01-20" lastUpdated="2025-09-29" className="mb-6" />

Next.js unterstützt das Exportieren eines `metadata`-Objekts aus Ihren MDX-Dateien, das verwendet werden kann, um SEO- und Social-Sharing-Funktionen zu verbessern. Dies ist besonders nützlich für Blog-Posts und Dokumentationsseiten.

## Beispiel: Metadaten exportieren

Sie können ein `metadata`-Objekt am Anfang Ihrer MDX-Datei wie folgt exportieren:

```javascript:app/blog/example-mdx-metadata/metadata.js
export const metadata = {
  title: 'Wie man Metadaten aus MDX für Next.js SEO exportiert',
  description: 'Ein Leitfaden zum Exportieren von Metadaten aus MDX-Dateien, um Next.js SEO-Funktionen zu nutzen.',
  authors: [{ name: 'Jane Doe', url: 'https://janedoe.com' }],
  openGraph: {
    images: [
      'https://cdn.cosmos.so/example-image.jpg',
    ],
  },
}
```

Diese Metadaten werden von Next.js erkannt und können für SEO, Open Graph und andere Meta-Tags verwendet werden.

## Warum Metadaten in MDX verwenden?

- **SEO**: Verbessern Sie das Ranking Ihrer Seite in Suchmaschinen mit relevanten Titeln und Beschreibungen.
- **Social Sharing**: Verbessern Sie, wie Ihr Inhalt beim Teilen auf Social-Media-Plattformen erscheint.
- **Konsistenz**: Halten Sie Ihre Metadaten nahe an Ihrem Inhalt für einfachere Verwaltung.

## Beispiel: TypeScript Konfiguration

```typescript:next.config.ts
import { withMDX } from '@next/mdx'

const nextConfig = {
  pageExtensions: ['js', 'jsx', 'mdx', 'ts', 'tsx'],
}

export default withMDX()(nextConfig)
```
